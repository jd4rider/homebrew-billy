# homebrew-billy

Homebrew tap for [Billy.sh](https://github.com/jd4rider/billy-app) — local AI coding assistant.

## Install

```bash
brew tap jd4rider/billy
brew install billy
```

## Upgrade

```bash
brew upgrade billy
```

## What is Billy?

Billy is a Copilot CLI alternative that runs entirely on your machine using [Ollama](https://ollama.com). No API keys, no subscription — just fast local AI.

Built with Go + [Bubble Tea](https://github.com/charmbracelet/bubbletea).
